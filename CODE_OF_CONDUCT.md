# Code of Conduct

## The Short Version

Be decent. Build things that help people. Assume good faith.

## The Longer Version

This project exists to build infrastructure for accountable systems. The same principles apply to how we work together.

### Expected Behaviour

- Be respectful and constructive in all interactions
- Focus on what's best for the project and its users
- Accept constructive criticism gracefully
- Give credit where it's due
- Help newcomers get oriented

### Unacceptable Behaviour

- Personal attacks or harassment of any kind
- Deliberately unconstructive criticism
- Publishing others' private information without consent
- Conduct that would reasonably be considered inappropriate in a professional setting

### Scope

This applies to all project spaces: GitHub issues, PRs, discussions, and any other channel where you represent the project.

### Enforcement

Instances of unacceptable behaviour can be reported to conduct@lovyou.ai. All reports will be reviewed and responded to appropriately.

### Attribution

This is a simplified code of conduct. It prioritises clarity over legal completeness. The spirit matters more than the letter.
